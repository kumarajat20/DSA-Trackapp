# 🚀 DSA Track App

A web application to track DSA progress, maintain streaks, and monitor performance.

## 🧠 Features
- User Authentication (Login/Signup)
- Track solved DSA problems
- Streak system
- Dashboard UI
- Dark/Light mode (if added)

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite / MySQL




## ⚙️ Setup Instructions

1. Clone the repo:
git clone https://github.com/your-username/DSA-Trackapp.git

2. Install dependencies:
pip install -r requirements.txt

3. Run app:
python app.py
